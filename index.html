Model Overview

The model is a scikit-learn Pipeline combining preprocessing and a Logistic Regression classifier. It was trained on a diabetes prediction dataset containing 100,000 patient records with 8 clinical and demographic features.

  • Algorithm: Logistic Regression (max_iter=1000)
  • Class imbalance handled with balanced class weights
  • Train / test split: 80% / 20%, random_state=42
  • Model auto-trains on first startup from the public dataset

Preprocessing Pipeline

  • Categorical features (gender, smoking_history) → One-Hot Encoding
  • Numeric features (age, BMI, HbA1c, blood glucose, hypertension, heart disease) → Standard Scaling
  • ColumnTransformer ensures both transformations run in parallel
  • Same transformations applied at training and inference time

About This App

A full-stack machine learning application built for demonstration purposes.

Application Overview

This application demonstrates a complete machine learning workflow — data preprocessing, model training, model persistence, and a professional web interface for real-time risk predictions. The backend is built with Flask and the ML pipeline uses scikit-learn.

Technology Stack

  • Backend: Python, Flask, Gunicorn
  • ML: scikit-learn (Logistic Regression, Pipeline, ColumnTransformer)
  • Data: Pandas, NumPy
  • Frontend: HTML, CSS, Vanilla JavaScript
  • Deployment: Render
